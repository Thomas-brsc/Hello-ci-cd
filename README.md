# Hello CI/CD

Ce projet est un exemple minimal de **pipeline CI/CD** utilisant GitHub Actions.

## Description

- Projet bash simple : `main.sh`  
- Pipeline CI/CD qui se déclenche **à chaque push**  
- Workflow minimal : affiche `Hello CI/CD` dans les logs  

## Contenu

- `main.sh` : script bash affichant `Hello CI/CD`  
- `.github/workflows/ci.yml` : pipeline CI/CD GitHub Actions
